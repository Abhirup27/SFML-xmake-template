# SFML-xmake-template

This is a simple template that uses xmake to build a project with SFML and C++. You do not need to have xmake, MSVC, mingw, SFML installed.

1. Open powershell
2. Run the desired command:

```
.\build.ps1 --(platform) --(build type)
```

## Platforms

- msvc
- mingw

## Build Types

- debug
- dev
- release
